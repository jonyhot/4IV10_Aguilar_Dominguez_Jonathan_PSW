habia una vez un patito que decia miau miau, y este patito queria aprender a utilizar diferentes tipos de formulario wiiii.
Existen dos metodos principales para la obtención de los datos dentro de un formulario web, para lo cual aqui lo explicaremos


GET:

Este metodo es el estandar como salida del navegador mediante el cual se resuelve una peticion por parte del cliente, asi mismo en la salida entandar (url), se puede visualizar los datos enviados mediante el formulario

POST:

Este es el metodo mas utilizado para el envio de información desde un formulario atendiendo una peticion por parte del cliente; actuando para poder agregar o insertar nuesvos datos en un BD, asi mismo a diferencia del metodo get se utiliza un tipo de seguridad midiante la cual la información no es visible por medio de la salida estandar del navegador
